## App Description

### Short Description (For App Store Preview)
Streamline your security operations with Harekar Group - the comprehensive workforce management solution designed specifically for security companies.

### Full Description

**Harekar Group** is a professional security workforce management application that revolutionizes how security companies track attendance, manage shifts, and coordinate their teams.

**Key Features:**

**Attendance Tracking**
- GPS-verified check-in and check-out with photo capture
- Real-time location verification to ensure guards are on-site
- Automatic late detection based on shift schedules
- Complete attendance history with timestamps and photos

**Shift Management**
- Create and assign shifts to security personnel
- Set custom shift times for each employee
- View daily shift schedules at a glance
- Track shift completion status in real-time

**Team Management (Supervisors)**
- Dashboard with live attendance statistics
- Monitor all security guards in real-time
- Add, edit, and remove team members
- View detailed employee profiles and attendance records

**Incident Reporting**
- Quick incident reporting with photo evidence
- Multiple incident categories (security breach, suspicious activity, equipment issues)
- Priority levels for urgent situations
- Complete incident history and tracking

**Announcements & Communication**
- Send instant announcements to all staff
- Priority messaging for urgent communications
- Pre-built message templates for quick sending
- Push notifications ensure messages are received

**Reports & Analytics**
- Daily, weekly, and monthly attendance reports
- Track attendance patterns and trends
- Export-ready data for payroll processing
- Performance insights for supervisors

**Built for Security Teams**
- Bilingual support (Kurdish Kurmanji & English)
- Intuitive interface designed for field use
- Works offline with automatic sync
- Secure data storage and transmission

Whether you're managing a small security team or a large enterprise operation, Harekar Group provides the tools you need to ensure accountability, improve communication, and streamline your security operations.
